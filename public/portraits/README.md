# Custom portraits

Drop 48×48 PNGs (or larger — any square works) here to replace the procedurally-drawn dialogue portraits.

Filename → replaces:

- `aida.png` — Aida, normal (colourful sweater)
- `aida-goth.png` — Aida in the black dress (Chapter I only)
- `paladin.png` — David, normal (band tee, beard)
- `paladin-white.png` — David in the white dress shirt (Chapter I only)
- `lisa.png` — Lisa
- `lisa-zombie.png` — zombie-Lisa (finale)

These portraits appear:
- Inside the bottom dialogue box (fallback when no speaker can be resolved)
- As the avatar on the phone chat overlay

Recommended: **48×48 or 64×64** PNG with transparency. Square is easiest.
